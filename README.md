# Midosis

App web para llevar seguimiento de medicamentos. Permite registrar tratamientos, configurar horarios de dosis y recibir recordatorios.

Desarrollada con React en el cliente y Node.js + Firebase en el servidor.

## Stack

| Capa | Tecnología |
|------|-----------|
| Frontend | React, JavaScript |
| Backend | Node.js, Firebase |
| Estilos | CSS |

## Funcionalidades

- Registro de medicamentos y dosis
- Configuración de horarios por tratamiento
- Historial de tomas
- Autenticación de usuario

## Ejecutar localmente

```bash
# Cliente
cd client
npm install
npm start

# Servidor
cd server
npm install
npm start
```
